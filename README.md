## アプリ概要

分・秒を入力してカウントダウンを行なうシンプルなタイマーアプリです。

タイマーの一時停止・再開や、ページリロード後の状態復元など、実用的な機能を備えています。

---

## 技術的なポイント

本アプリでは、タイマーの精度改善を目的として「終了時刻ベース」のロジックを採用しています。

通常のタイマーは`1秒ごとに数値を減らす方法`が多いのですが、この方法では以下の問題が発生します。

・タブ非アクティブ時の遅延
・リロード時のズレ
・setInterval の誤差

これらを解消するために、`「現在時刻」と「終了時刻」の差分から残り時間を算出する方法`に変更しました。

## Features（主な機能）

- カウントダウンタイマー機能（分・秒入力）

- 時間のズレを防ぐ仕組み（終了時刻ベース）

## 工夫した点

・`timer !== null` を利用し、``Startボタンの連続クリックによるタイマー重複起動を防止``しています。

・Stop後に再度Startした場合、途中から再開できるように状態を保持

・localStrageのデータが破損していた場合でもアプリが停止しないために、`try-catch` による``例外処理``を実装

・localStorage を用いて、ページリロード後もタイマーの状態（残り時間・動作状態・入力値）を復元

・時間ズレを防ぐため、ミリ秒単位で計算を行なうように改善
