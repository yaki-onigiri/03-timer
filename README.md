## Features（主な機能）

2026/03/17

- スタート機能
「Start」ボタンを押すと1秒ごとにカウントアップします。

- ストップ機能
「Stop」ボタンを押すとタイマーを一時停止できます。

- リセット機能
「Reset」ボタンを押すとタイマーを 00：00 に戻します。

- 時間表示機能
秒数を「分：秒」の形式に変換して表示します。
（例：01：05　など）

2026/03/18

- カウントダウンタイマー機能
分：秒を入力してカウントダウンタイマーを開始できます。

- Start / Stop / Reset の基本操作

- タイマーの一時停止および途中からの再開
一度止めたタイマーをStartボタンを押すことで再開できるようにしました。

- 0秒になった時点で自動停止
- 0秒になった時点でアラーム音を自動的にならす
設定した時間が切れたときに、アラーム音を鳴らしてユーザーに知らせる機能を追加しました。

・HTML

## 工夫した点

26/03/17
・秒数を“そのまま表示する”のではなく、「分：秒」形式で表示する処理を実装しました。

・`padStart()` を使用して、``常に2桁で表示される``ように調整しました。

・`timer !== null` を利用し、``Startボタンの連続クリックによるタイマー重複起動を防止``しています。

26/03/18
・Startボタンを複数回押してもタイマーが重複起動しないように制御しました。

・if (seconds === 0) の条件分岐を使い、初回のみ入力値を反映するようにしました。

・Stop後に再度Startした場合、途中から再開できるように上体を保持するようにしました。
